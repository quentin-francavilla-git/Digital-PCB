II - Lancer le projet

1. Ouvrir CATIA
2. Ouvrir le dossier "Digital-PCB-main"
3. Clique sur la barre avec le chemin du dossier
4. Copier le chemin (exemple : C:\Users\Quentin\Documents\Digital-PCB-main)
5. Touche Windows
6. écrire :
    cmd
7. écrire :
    cd C:\Users\Quentin\Documents\Digital-PCB-main
8. écrire
    python app.py
9. Se rendre sur 
    http://127.0.0.1:5000
